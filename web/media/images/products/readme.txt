Product images are structured in the following way:

- One folder per Product, the name of the folder is [mfg]-[name]-[variant]
- MAIN IMAGE - Each folder must contain a main.jpg which is the primary, touched up image for the product
- MAIN IMAGE SCALED - Each folder must contain a main-300.jpg and main-600.jpg which are main imaged scaled to 300px and 600px respectively.
- OTHER IMAGES - Each folder may contain additional images of the product for the purpose of present in a carousel or gallery fashion.  Names should be carousel-001.jpg, carousel-002.jpg etc
- PRODUCT SHEETS (PDF) - Each folder may contain a product-sheet.pdf
